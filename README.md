# Mi Portafolio Personal

Este es mi portafolio personal desarrollado con tecnologías modernas de desarrollo web. El sitio web está diseñado para mostrar mis proyectos, habilidades y experiencia profesional.

## 🚀 Tecnologías Utilizadas

- **React 18** - Biblioteca de JavaScript para construir interfaces de usuario
- **Vite** - Herramienta de construcción moderna y rápida
- **Tailwind CSS** - Framework CSS para diseño responsivo
- **Framer Motion** - Biblioteca para animaciones
- **EmailJS** - Servicio para manejo de formularios de contacto
- **GitHub Pages** - Plataforma de hosting para el sitio web

## 📋 Características

- Diseño responsivo y moderno
- Animaciones suaves con Framer Motion
- Formulario de contacto funcional
- Soporte para múltiples idiomas
- Optimizado para SEO
- Despliegue automático con GitHub Pages

## 🛠️ Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/lokogam/Duvan-Gamboa.git
```

2. Instala las dependencias:
```bash
npm install
```

3. Inicia el servidor de desarrollo:
```bash
npm run dev
```

## 📦 Scripts Disponibles

- `npm run dev` - Inicia el servidor de desarrollo
- `npm run build` - Construye la aplicación para producción
- `npm run preview` - Previsualiza la versión de producción
- `npm run lint` - Ejecuta el linter
- `npm run deploy` - Despliega la aplicación a GitHub Pages

## 🌐 Despliegue

El sitio está desplegado en GitHub Pages y se actualiza automáticamente cuando se hace push a la rama principal.

URL del sitio: https://lokogam.github.io/Duvan-Gamboa
